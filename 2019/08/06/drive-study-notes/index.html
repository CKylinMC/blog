学车笔记

发表于 2019-08-06 更新于 2019-08-08 分类于 学习 阅读次数：

学车第二天开始留下的笔记。

基础

上车步骤

  1. 先调座椅，要求左脚离合踩到底依然有弯曲，双手手腕可以平放于方向盘上。
  2. 系安全带。
  3. 调整后视镜。

侧镜调整

  • 两侧车门把手在侧镜里侧，上下处于中间的位置。
  • 另一种方法是右侧镜子为里侧宽1指，顶部宽1指的点对准右侧车门后上角，左侧是顶部2指。

点火起步

标准平路起步

  • 看发动机转速指针是否有读数，没有就是没启动，有就是已启动。
  • 如果已启动再点火直接不及格。
  • 点火时间超过2s扣10分。
  • 踩下离合，空档点火，发动机有读数，车身一个明显震动后松手。
  • 挂档，松手刹，轻抬离合可起步。

爬坡起步

  • 点火同上。
  • 不松手刹，挂档，慢松离合直到车头上抬，离合不要再动，松手刹起步。

定点停车起步（油门起步）

  1. 点火同上。
  2. 拉手刹，踩死离合。
  3. 踩死油门，慢松离合，直到车头上抬，发动机读数20，松手刹猛起步。
    • [2019-8-7] 踩死油门，慢松离合，车头上抬2~3次，声音改变并发闷，推荐发动机读数20，可保持状态并松手刹起步。
  4. 速度稳定后松油门，然后松离合。

要点

技术动作

左右倒车入库

上线

  1. 在线外，黄线于身体对齐。
  2. 停车，右打一圈进入。
  3. 当左镜左侧距离左边线约一拳时停车，左打两圈（等于从正向左打一圈）。
  4. 车辆平行时回正，向前压线，执行右倒库、左倒库，然后回到此处。
  5. 向后退足够距离，左侧离开。
  6. 全程应不超过210秒（3分30秒）。

出库

  1. 向前轻抬离合，车头左侧鼓起压白线可以打轮打死。
  2. 转完后回轮，粗黄线平行于方向盘下方位置停车。
  3. 倒车时左镜下边缘与粗黄线视线重合时可打轮打死。

入库

  • 车左侧镜与白线内边缘视线重合时立刻刹车。

左入库

  • 压住第三条虚线的2/3后回一圈，旁边的垂直角消失时再打死，车身回正后方向盘归位。

右入库

  • 压住第三条虚线一个角后回一圈，旁边的垂直角消失时再打死，车身回正后方向盘归位。

侧方入库

  1. 直行到右侧镜看到黄色标点位于中偏上位置停车。
  2. 后退，拐角和虚线之间第一个断开处（黑色线）消失时右打死。
  3. 探身提前观察左侧镜子，看到车库右后边角线时立刻回正。
  4. 探身看车轮压虚线（个人经验：推荐车轮斜后方下45度指向位置压住虚线）时左打死。
  5. 右侧镜子看车身平行时不改方向盘。只停车。
  6. 开左侧转向灯，向前挂档。
  7. 左车盖突起碰到路左线方向盘回正。
  8. 路左线到车盖中间位置时右一圈，车身平行时回正。

定点停车

  1. 方向盘提前对准从右往左第二条虚线左边或箭头。
  2. 车左侧镜下边看到白线立刻停车，车头可在黄线内。
  3. 使用油门起步离开。
  4. 压线不及格，车身距离路边超出30cm扣10分，车头在黄线内不扣分，否则扣10分。

S型弯道

  1. 离合控制车速
  • 进入弯道时，线过车头一半时打右侧一圈。
  • 当右侧线过一半时方向盘回正，过左突起时左侧一圈多一点。
  • 左侧线过车头一半以上时右打轮，出弯道。
  • [2019-8-7]
    • 入向右拐弯的弯道时，用雨刮器根部对左线时，向右打轮一圈零一刻，雨刮器根部对右线时回正。
    • 入向左拐弯的弯道时，用左突起对右线时，向左打一圈零一刻，左突起根部对左线时回正。

直角弯道

  1. 开左转向灯，听到提示后关闭。
  2. 方向盘对准箭头。
  3. 内侧直角线与方向盘[9点方向]平行时左打死，出弯。
